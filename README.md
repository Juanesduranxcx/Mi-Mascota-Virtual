- Mi Mascota Virtual - Java

¡Bienvenido a mi proyecto de Mascota Virtual!

Este es un juego interactivo desarrollado en **Java** corriendo por consola, donde el usuario puede registrar su mascota y gestionar sus estados básicos.

[] Características
* **Sistema de turnos:** El juego evalúa las acciones del usuario de forma dinámica.
* **Acciones disponibles:** Puedes alimentar, jugar o poner a dormir a tu mascota.
* **Gestión de estadísticas:** El sistema controla los niveles de energía y hambre en tiempo real.

[] Tecnologías utilizadas
* **Java SE** (Programación Orientada a Objetos)
* **Scanner Utility** (Para la interacción con el usuario en consola)
